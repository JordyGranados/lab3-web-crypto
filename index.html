¿Cómo funciona?

Este laboratorio usa el algoritmo AES (Advanced Encryption Standard) a través de la librería CryptoJS. El texto se cifra con una llave secreta y sólo con esa misma llave puede ser descifrado.

⚠️
Nota de seguridad

Para producción real usa llaves de 256 bits generadas aleatoriamente y almacenadas de forma segura (p.ej. variables de entorno VITE_APP_KEY).
